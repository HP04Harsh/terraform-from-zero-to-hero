# Terraform: From Zero to Hero 🚀

Welcome to my learning journey! This repository is designed to document my progress in mastering **Terraform**, covering everything from basic syntax to advanced orchestration.

## 📚 Learning Path
- [ ] **Fundamentals:** Providers, Resources, and basic HCL syntax.
- [ ] **State Management:** Understanding `terraform.tfstate`, backends, and locking.
- [ ] **Data Handling:** Variables, Outputs, and Data Sources.
- [ ] **Logic & Control:** Using `count`, `for_each`, and conditional logic.
- [ ] **Modularization:** Creating reusable modules and directory structures.
- [ ] **Advanced:** Provisioners, Workspaces, and Sentinel/Policy as Code.

## 📂 Repository Structure
Each folder represents a specific concept or lab:
* `01-getting-started`: The absolute basics.
* `02-state-management`: Deep dive into state files.
* `03-advanced-patterns`: Complex module configurations.

## 🛠 How to Use This Repo
1. **Clone the repo:** `git clone https://github.com/HP04Harsh/terraform_basic_to_advanced.git`
2. **Navigate to a module:** `cd 01-getting-started`
3. **Run your tests:**
   ```bash
   terraform init
   terraform plan
   terraform apply
   ```
